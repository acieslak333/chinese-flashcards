# Chinese Flashcards

A modern, interactive Chinese flashcard application built with React and Tailwind CSS. Designed for efficient vocabulary retention with a focus on aesthetics and user experience.

## Features

- **Smart Learning Modes**:
  - **Spaced Repetition**: Mark cards as Easy, Medium, or Hard to customize your review flow.
  - **Random Blur**: Test your recall by randomly hiding Chinese characters, Pinyin, or meanings.
  - **Example Sentences**: Toggle usage examples for deeper context.

- **Powerful Filtering**:
  - **Chronological Sorting**: Lessons sorted by date (DD.MM.YYYY).
  - **Multi-Select Filters**: Filter by specific years, months, or difficulty levels.
  - **Search**: Quickly jump to specific lessons.

- **Aesthetic Themes**:
  - **Modern Collection**: Matcha Latte, Light Blue Sky, Soft Ruby, Neon City, End of Summer.
  - **Retro Collection**: Retro 90s (Windows 95 style), Y2K Cyber (Early 2000s tech).
  - **Basics**: Classic White, Midnight Black, Soft Beige, Deep Navy.

- **Smooth Experience**:
  - **Flat UI**: Clean, shadow-free design with ultra-rounded elements.
  - **Animations**: Smooth transitions for all interactions (reveals, navigation, settings).
  - **Shortcuts**: Full keyboard support for rapid reviewing.

## Getting Started

1.  **Install Dependencies**: `npm install`
2.  **Start App**: `npm start`
3.  **Build**: `npm run build`

## Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
